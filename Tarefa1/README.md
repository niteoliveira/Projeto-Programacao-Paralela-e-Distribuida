"# Tarefa 1: Implementação Básica com Deadlock

## Descrição

Implementação básica do problema do Jantar dos Filósofos que **pode causar deadlock**. Todos os filósofos pegam o garfo esquerdo primeiro, depois tentam pegar o direito.

## Como Executar

```bash
javac *.java
java Main
```

## Problema de Deadlock

Deadlock ocorre quando todos os 5 filósofos pegam simultaneamente o garfo esquerdo e ficam esperando eternamente pelo garfo direito, que nunca ficará disponível.
